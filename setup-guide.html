ğŸ”— API Endpoints

Authentication

  • POST /api/register - Register new user
  • POST /api/login - User login

User Data

  • POST /api/userdata - Save new data
  • GET /api/userdata/:userId - Get user's data
  • PUT /api/userdata/:id - Update data entry
  • DELETE /api/userdata/:id - Delete data entry

Admin

  • GET /api/admin/users - Get all users with data
  • GET /api/admin/users/:userId - Get user details
  • DELETE /api/admin/users/:userId - Delete user

ğŸ› Troubleshooting

Server won't start?

  • Make sure Node.js is installed (v14+)
  • Port 3000 might be in use - change it in server.js
  • Delete database.db if corrupted and restart
  • Run: npm install again

Login fails?

  • Check if server is running on http://localhost:3000
  • Open browser console (F12) and check for errors
  • Try the default admin account first
  • Clear browser cache and refresh

Can't save data?

  • Ensure title field is filled (it's required)
  • Check browser console for errors
  • Verify backend server is running
  • Check database permissions
